Bienvenue sur le démonstrateur de l'API ISTEX !

Ce démonstrateur a une vocation pédagogique pour montrer les différents aspects de l'API ISTEX.

Le code source du site peut cependant servir comme base de réflexion pour construire ou améliorer un portail, comme le font les widgets ISTEX. Ceux-ci permettent de construire rapidement des interfaces graphiques permettant d'interroger le fond ISTEX.

Ce démonstrateur illustre les différents aspects de l'API ISTEX, de la recherche à l'affichage des résultats, en passant par l'utilisation des facettes. Des infobulles sont disponibles sur l'ensemble des fonctionnalités pour vous guider.

Ce démonstrateur ne constitue donc pas un portail documentaire sur l'API.

Pour toute aide supplémentaire ou suggestion, vous pouvez nous écrire à istex@inist.fr.
